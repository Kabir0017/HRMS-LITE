# HRMS Lite – Full Stack Assignment

## Overview
HRMS Lite is a lightweight Human Resource Management System that allows an admin to:
- Manage employee records (add, view, delete)
- Track daily attendance (Present / Absent)

This project demonstrates end-to-end full‑stack development including frontend, backend, database integration, and deployment.

## Live Links
- **Frontend (Vercel):** https://hrms-lite-topaz-seven.vercel.app/  
- **Backend API (Render):** https://hrms-lite-jap7.onrender.com  
- **API Docs (Swagger):** https://hrms-lite-jap7.onrender.com/docs  
- **GitHub Repo:** https://github.com/Kabir0017/HRMS-LITE  

## Tech Stack
**Frontend**
- React  
- Axios  
- CSS  

**Backend**
- FastAPI (Python)  
- SQLAlchemy  

**Database**
- SQLite  

**Deployment**
- Frontend: Vercel  
- Backend: Render  

## Features
- Add, list, and delete employees  
- Mark daily attendance (Present / Absent)  
- Dashboard summary (Total Employees)
